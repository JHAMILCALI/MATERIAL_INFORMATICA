Navegador lateral fijo de altura completa

Intente desplazarse por esta área y vea cómo la navegación lateral se adhiere a la página.

Tenga en cuenta que este elemento div tiene un margen izquierdo del 25%. Esto se debe a que la navegación lateral está configurada en un 25 % de ancho. Si elimina el margen, el sidenav se superpondrá/se asentará encima de este div.

También tenga en cuenta que hemos configurado overflow:auto en sidenav. Esto agregará una barra de desplazamiento cuando la navegación lateral sea demasiado larga (por ejemplo, si tiene más de 50 enlaces dentro).

La propiedad overflow de CSS controla lo que sucede con el contenido que es demasiado grande para caber en un área.

Un elemento con position: fixed;se coloca en relación con la ventana gráfica, lo que significa que siempre permanece en el mismo lugar, incluso si se desplaza la página

La propiedad display especifica el comportamiento de visualización (el tipo de cuadro de representación) de un elemento.

not; El selector coincide con todos los elementos que NO son el elemento/selector
